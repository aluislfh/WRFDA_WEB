Get pre-compiled code

To save time and avoid unnecessary errors, we recommend using the provided pre-compiled code to run the exercises.

The pre-compiled WRFDA code can be accessed from /wrf2017/WRFDA.

You should un-tar the pre-compiled code package into your local working directory. Follow the steps below:

Pre-compiled code for 3DVAR

mkdir /wrf2017/groups/${USER}/DA    (your main working directory)
cd /wrf2017/groups/${USER}/DA
tar -xzf /wrf2017/WRFDA/WRFDA_compiled.tar.gz

This will generate the WRFDA directory containing the compiled WRFDA code. This WRFDA package is pre-compiled to run in parallel (dmpar) mode using the GNU compiler (gfortran/gcc 5.3.1).
