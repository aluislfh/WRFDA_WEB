WRFDA Practice Sessions

GETTING STARTED: After you have logged on and opened a terminal window (Applications --> System Tools --> Terminal), change directory to the main working directory for WRFDA tutorial practice:

cd /kumquat/users/${USER}/DA

If the directory does not exist yet, you should create it.

mkdir /kumquat/users/${USER}/DA

 

Refer to the WRFDA User's Guide if you need more detailed descriptions on various components of the WRF Data Assimilation System.

Tutorial presentations can be accessed through the links on each session's page.

The exercises are organized in the order each topic is introduced. For each exercise, you will have a basic practice with preset namelists and step-by-step commands to guide the first-time users through the procedures. Most exercises also contain additional practice listed in the end of each page; these exercises contain less instruction, so users will need to refer to documention and previous lessons to figure out how to accomplish the tasks.

These practice pages are designed for the Summer 2016 WRFDA tutorial at NCAR Foothills lab. The path names on this web site are tailored for the NCAR classroom environment. The path names that start with a forward slash / are absolute paths and they are specific to the classroom computers. The paths that do not start with a slash are relative paths. Users need to fill in or change the full path names when necessary.
