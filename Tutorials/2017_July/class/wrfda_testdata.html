WRFDA practice test datasets

 

Links will be posted here at the end of the tutorial so you can download the test data for practice at your home institution.

From this page, you can download the test data used in the classroom exercises. See the download links below.

For NCL scripts from the WRFDA TOOLS package (used in some exercises), see this page.

For the practice sessions, to save data-copying time, we recommend you use the files that are already un-zipped and un-tarred under /kumquat/wrfhelp/DATA/WRFDA directory.
