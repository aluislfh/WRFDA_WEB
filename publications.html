WRFDA System Papers:

As a professional courtesy, we ask that any publications which make use of any part of the WRFDA system cite the appropriate reference(s) from the following list. In general, these are the original publications which describe each capability listed, but in some cases they represent an overview of the given topic:

WRFDA system as a whole
