About WRFPLUS

WRFPLUS is a package containing the adjoint and tangent-linear version of WRF model, in addition to a specialized version of the non-linear WRF model (previously known as WRFNL). It is tailored for use with WRFDA 4D-Var only. You should use the same version number of WRFPLUS as you do WRFDA.

The most recent WRFPLUS build is Version 3.9, released on April 17, 2017. It is consistent with the WRF V3.9 release, and should be used with WRFDA-4DVAR V3.9. To learn more about new features and fixes included in the most recent WRFDA/WRFPLUS update, see the update page.

How to build WRFPLUS:

  • Download the desired WRFPLUS archive
  • gunzip <filename>.tar.gz
  • tar -xvf <filename>.tar
  • cd WRFPLUSV3
  • ./configure wrfplus
  • ./compile em_real

For more detailed instructions, see the User's Guide.

For older versions, see the WRFNL page. Please note that WRFNL is no longer supported.
