Forecast Sensitivity to Observations

Forecast Sensitivity to Observations (FSO) is a package designed to compute the impact of observations on the forecast error reduction when used with the WRF ARW (Advanced Research WRF) dynamical core and WRFDA.

Due to limited resources, only limited support is available for this utility.

Documentation

The .tar package available below contains a README file with installation instructions. FSO requires the WRFDA TOOLS package, which can be downloaded from the link in the side bar. Additional guidance is also available from the FSO V3.3 User's Guide, and the Summer 2014 tutorial presentation, but please note that some of this information may be out-of-date.
